# Creating users in Thunder using the console

*This document was created by an AI agent following the Thunder documentation creation instructions.*

## Purpose Hold

This guide explains how to create users in Thunder using the console application. It targets system administrators and IT professionals who manage user accounts.

## When to use This Provide

Use this guide when you need to add new users to Thunder for authentication and authorization purposes. This is applicable when setting up a new system or managing existing deployments.

## Prerequisites And

Before you begin, ensure the following:

- You have access to the Thunder console application.
- You have the necessary permissions to create users.
- You know the required user details, such as name, email, and roles.

## Steps

1. Log in to the Thunder console application.
   - Open the Thunder console in your web browser.
   - Enter your administrator credentials and select **Log In**.

2. Navigate to the **Users** section.
   - From the main menu, select **Users**.

3. Click **Add User**.
   - Locate the **Add User** button and click it.

4. Fill in the user details.
   - Enter the required information, such as:
     - **Name**: The full name of the user.
     - **Email**: The user's email address.
     - **Role**: Assign the appropriate role to the user.
   - Ensure all mandatory fields are completed.

5. Save the user.
   - Click **Save** to create the user.

## Validate

To confirm the user was created successfully:

- Check the **Users** list to ensure the new user appears.
- Verify the user details, such as name and email, are correct.

## Troubleshoot

If the user creation fails:

- Ensure you have the necessary permissions to create users.
- Verify that all mandatory fields are filled in correctly.
- Check for error messages and resolve any issues mentioned.
- Contact your system administrator if the problem persists.

## Next steps

- Assign additional roles or permissions to the user if required.
- Notify the user about their account and provide login instructions. Learn more about managing users in Thunder by visiting the [User Management Guide](/guides/user-management).
